# HTML Basic Tags

This project demonstrates the use of basic HTML tags for beginners learning web development.

## 📋 Tags Included

* Heading Tags (`<h1>` to `<h6>`)
* Paragraph Tag (`<p>`)
* Anchor Tag (`<a>`)
* Image Tag (`<img>`)
* List Tags (`<ul>`, `<ol>`, `<li>`)
* Table Tags (`<table>`, `<tr>`, `<th>`, `<td>`)
* Form Tags (`<form>`, `<input>`, `<button>`)
* Division Tag (`<div>`)
* Span Tag (`<span>`)
* Line Break Tag (`<br>`)
* Horizontal Rule Tag (`<hr>`)

## 🚀 Features

* Simple and beginner-friendly HTML structure
* Demonstrates commonly used HTML elements
* Easy to understand and modify

## 🛠️ Technologies Used

* HTML5

## ▶️ How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
3. Explore the basic HTML tags and their output.

## 📚 Learning Purpose

This project is created for practicing and understanding the fundamental HTML tags used in web development.
